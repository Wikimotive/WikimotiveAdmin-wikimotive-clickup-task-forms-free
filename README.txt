== Installation ==

Installation Instuctions:
1. Download and install the plugin from the Wordpress Plugins Repository located here:
   - or - By downloading the zip file from the Github Repository Located Here: https://github.com/Wikimotive/wikimotive-clickup-task-forms-free, then install the plugin from the manual install within the Plugins menu in your wordpress admin
2. Activate the plugin
3. For configuration instructions, please see the "Help" page in the ClickUp Forms submenu
