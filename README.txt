This plugin allows you to add Task Submission Forms for ClickUp to your Wordpress website via the use of shortcodes and ClickUp's Cloud API Connection.

== Description ==

### Placing the form on your site:
Once you've created your form you'll be able to place it anywhere on your site with the use of the shortcode below. 
[ctf_form id="{Form ID}"]

== Installation ==

Installation Instuctions:
1. Download and install the plugin from the Wordpress Plugins Repository located here: https://wordpress.org/plugins/wikimotive-clickup-task-forms-free/
   - or - By downloading the zip file from the Github Repository Located Here: https://github.com/Wikimotive/wikimotive-clickup-task-forms-free/, then install the plugin from the manual install within the Plugins menu in your wordpress admin
2. Activate the plugin
3. For configuration instructions, please see the "Help" page in the ClickUp Forms Wordpress Admin Submenu
